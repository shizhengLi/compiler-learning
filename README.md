# Compiler Learning Project

一个完整的编译器学习项目，包含技术博客和实际实现的编译器组件。

## 项目结构

```
compiler-learning/
├── tech-blog/          # 技术博客文章
├── src/               # 编译器源代码
│   ├── lexer/         # 词法分析器
│   ├── parser/        # 语法分析器
│   ├── semantic/      # 语义分析器
│   ├── codegen/       # 代码生成器
│   ├── optimizer/     # 代码优化器
│   └── runtime/       # 运行时系统
├── tests/             # 测试用例
├── examples/          # 示例程序
└── README.md          # 项目说明
```

## 学习路径

1. 阅读 tech-blog/ 中的技术博客文章
2. 查看 src/ 中对应组件的实现
3. 运行 examples/ 中的示例程序
4. 在 tests/ 中编写测试用例

## 技术博客目录

01. 编译器架构概览
02. 词法分析与符号化
03. 语法分析与解析
04. 语义分析与符号表
05. 中间代码生成
06. 代码优化技术
07. 目标代码生成
08. 运行时系统与内存管理
